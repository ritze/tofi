tofi
====

Print TODOs, FIXMEs & Co.

![Screenshot](screen.png)

Supported types:
* ```BUG```
* ```FIXME```
* ```HACK```
* ```NOTE```
* ```OPTIMIZE```
* ```TODO```
* ```XXX```

Supported comment styles:
* ```//```
* ```#```
* ```''```
* ```""```
* ```;```
* ```--```
* ```/* … */```
* ```/** … **/```
